Code Translation
The code is translated from Kotlin to Swift and Typescript, resulting in an easily readable copy of your program for the destination platform.

Direct Integration
Because your code is directly translated, you can easily use platform-specific libraries. Just write a file for each non-Android platform describing the equivalents for the other platforms!
